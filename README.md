# hackathon-tank

core/以下に画像、入力系関数があります

## resource.js
まずloadImage("キーにする名前","画像へのパス")で画像を読み込んで
drawImage("画像のキー",x,y)で描画できます

drawRotatedImage("画像のキー",画像の中心を合わせるx,y,角度)で角度つきの描画もできます
ただし中心を合わせるように配置するので注意！
